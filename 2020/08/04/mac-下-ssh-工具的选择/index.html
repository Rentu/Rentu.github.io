mac 下 ssh 工具的选择

2020-08-04

win 下一直用xshell，转到mac 下，之前是用iterm2，今天有点闲暇，尝试了一下其他相关软件，最终还是选择了Iterm2

尝试软件如下：

  • ssh shell 评价太低，下载都懒得下了
  • Royalist 流程复杂，而且不支持google的二次验证，重点是我有连接docker服务器的需求，一般 docker exec -it xxx /bin/bash 这种，这软件还给我打开mac默认的终端且不能配置
  • electerm 开源的，也不支持二次验证，算了

为啥最终还是选择了iterm2，最后我发现，可以通过写脚本实现保存密码和二次登陆并且有很强的扩展性，装上zsh，也是非常好看的, 效果图来一发，巴适。

脚本就是利用except实现一些简单的交互, except 需要单独安装

brew install expect

嗯，自己动手，丰衣足食。
